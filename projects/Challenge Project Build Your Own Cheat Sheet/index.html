Tag Name Description
ie: h1, h2, .class, html, footer, etc. Selector The beginning of the ruleset used to target the element that will be styled.
ie: h1 { background-color: red;} Declaration Block The code in between (and including) the curly braces that contains the CSS declaration.
ie: background-color: red;, color: blue;, font-size: 20px;, font-family: Arial;, font-weight: bold;, etc. Declaration The group name for a property and value pair that applies a style to the selected element.
ie: background-color, color, font-size, font-family, font-weight, etc. Property The first part of the declaration that signifies what visual characteristics of the element is to be modified.
ie: red;, blue;, 20px;, Arial;, bold;, etc. Value The second part of the declaration that signifies the value of the property.
